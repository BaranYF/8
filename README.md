The Concept behind this project was to show renewable energy in a more modern way, by drawing inspiration from Apple and another renewable energy org NextEra Energy. I tried to make it large, full width images and content inside one single page, with smooth and easy scrolling. by using navigation buttons i tried to make it as accesible/easy access as possible to get to specific information that you are after. transparency was my main design choice with rounded edges. i tried to make the hover effect in javascript but got in problems and it took to much of my time so i did it in css. the contact form is still on the page and i made it into an overlay to keep the design ruke of keeping you as much as possible on one page.
